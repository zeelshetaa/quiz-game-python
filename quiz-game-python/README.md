# Python Quiz Game

A simple command-line Quiz Game built with Python.

## Features

- Multiple-choice quiz questions
- Score calculation
- Countdown timer
- Clean modular project structure
- Beginner-friendly code

## Project Structure

quiz-game-python/
│
├── main.py
├── questions.py
├── quiz_logic.py
├── timer.py
├── requirements.txt
└── README.md


## How to Run

1. Clone the repository

git clone https://github.com/yourusername/quiz-game-python.git

2. Navigate into project folder

cd quiz-game-python

3. Run the program

python main.py
